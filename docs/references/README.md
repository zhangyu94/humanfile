# References

This directory is for reference material derived from sources outside this repository.

Use this folder for:
- Summaries of external standards/specifications.
- Notes from vendor/platform documentation.
- External syntax references used by implementation/docs.

Do not use this folder for:
- Internal project execution plans or plan history.
- Normative project contracts (use `docs/specs/`).
- Internal design decisions (use `docs/design/`).
